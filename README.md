# SpiderWave
OPL4 cartridge for MSX computers. MoonSound compatible. 

## Specification

* Yamaha YMF278B OPL4 sound chip 
* NXP Philips UDA1334ATS DAC
* 2 MB Sample RAM
* 2 MB Yamaha YRW-801 ROM 
* External stereo audio output (TRS 3.5mm jack)
